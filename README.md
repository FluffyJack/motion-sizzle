# MotionSizzle

View selector engine based off jQuery's Sizzle Selector Engine. Use CSS-like selectors to find your views.

## Installation

Add this line to your application's Gemfile:

    gem 'motion-sizzle'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install motion-sizzle
